# راهنمای مگا سوپرادمین

## نقش مگا سوپرادمین چیست؟
مگا سوپرادمین تنها کاربری است که دسترسی کامل و سراسری به پنل Admin دارد و می‌تواند سازمان بسازد، پلن‌ها را تعریف/ارتقا/دانگرید کند و محدودیت‌ها را مدیریت کند.

## مسیر پیشنهادی کار (سریالی)
1) **ایجاد سازمان** از منوی «ایجاد سازمان (ویزارد)»
2) انتخاب **سوپرادمین سازمان**
3) انتخاب **ماژول‌های فعال** برای سازمان
4) تعیین **پلن/سهمیه‌ها** (تعداد فضای کاری، تعداد کاربر، وضعیت فعال/آزمایشی)
5) ساخت **فضای کاری اولیه** و تعیین مالک آن

## قواعد مهم
- تنها ایمیل‌هایی که در تنظیمات `FILAMAT_IAM_MEGA_SUPER_ADMINS` قرار گرفته‌اند، مگا سوپرادمین هستند.
- پلن سازمانی اگر **inactive/expired** شود، فقط دسترسی‌های پایه (مشاهده/مدیریت اشتراک) باقی می‌مانند.

## ارتقا/دانگرید
- تغییر پلن با به‌روزرسانی entitlements سازمان انجام می‌شود.
- بعد از تغییر پلن، همگام‌سازی نقش‌ها به‌صورت خودکار انجام می‌گیرد.

## نکات عملیاتی
- اگر سازمان به سقف فضای کاری رسید، ایجاد workspace جدید ممکن نیست.
- هر تغییر در entitlements باید از پنل Admin انجام شود.
