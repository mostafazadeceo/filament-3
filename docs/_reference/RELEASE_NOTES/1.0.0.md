# یادداشت انتشار 1.0.0

تاریخ: 2026-01-03

## خلاصه
- انتشار پایه پلتفرم Haida با معماری ماژولار Filament v4 و پوشش کامل چند‌مستاجری و IAM.

## موارد مهم
- پنل Admin و Tenant با پلاگین‌های اختصاصی ماژول‌ها.
- IAM Suite با تیم‌بندی spatie و قابلیت‌های scoped API.
- ماژول‌های هسته (Commerce، Workhub، Payments، Providers، CMS، Blog، Loyalty، Crypto، POS، Accounting/Payroll IR).
- زیرساخت اعلان‌ها و وبهوک‌های امن با idempotency.
- مستندات مرجع و Runbook عملیاتی.

## مهاجرت‌ها
- اجرای کل مهاجرت‌ها برای IAM و ماژول‌ها الزامی است: `php artisan migrate --force`.

## نکات عملیاتی
- برای همگام‌سازی قابلیت‌ها: `php artisan filamat-iam:sync --guard=web`.
- OpenAPI هر ماژول از مسیر `/api/v1/<module>/openapi` در دسترس است (در صورت وجود).
