# ممیزی رگرسیون (Regression Audit)

## هدف
اطمینان از عدم رگرسیون در ماژول‌های کلیدی پس از هر انتشار.

## اجرای سریع (Smoke)
```bash
APP_ENV=staging ./scripts/regression-smoke.sh
```

## پوشش سناریوهای کلیدی
- حسابداری: ثبت سند متوازن + صدور سند فروش/خرید
- انبار: ثبت رسید/حواله و کنترل موجودی منفی
- کیف پول: افزایش موجودی + پرداخت سفارش
- کامرس: سفارش/پرداخت/ثبت خروجی انبار
- دامنه‌ها: درخواست/تأیید دامنه + TLS request
- Provider: همگام‌سازی محصولات + dead-letter reprocess
- Workhub: ایجاد آیتم و انتقال وضعیت

## یادداشت‌ها
- اجرای Smoke از SQLite استفاده می‌کند تا محیط‌های اصلی درگیر نشوند.
- اجرای این سند روی production ممنوع است.
