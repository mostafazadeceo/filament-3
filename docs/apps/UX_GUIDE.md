# UX Guide (Persian-first)

## اصول
- RTL کامل در همه صفحات.
- فونت فارسی (Vazirmatn برای Web).
- تاریخ جلالی در نمایش‌ها.

## ناوبری
- منو بر اساس permission فعال/غیرفعال شود.
- وضعیت sync همیشه قابل مشاهده باشد.

## UX آفلاین
- وضعیت آفلاین واضح + صف outbox نمایش داده شود.
- عملیات حساس (پرداخت‌های غیرنقدی) در حالت آفلاین محدود یا staged.

## رنگ و تایپوگرافی
- رنگ‌های اصلی: teal + orange.
- کنتراست مناسب برای متن‌ها و هشدارها.

## الگوهای تعامل
- Actions مهم با confirmation.
- اعلان‌ها کوتاه و بدون اطلاعات حساس.
