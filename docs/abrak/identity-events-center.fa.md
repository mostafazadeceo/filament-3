# هاب ابرک به‌عنوان مرکز هویت و رویداد

## نقش
- مرجع مرکزی احراز هویت واحد (SSO)
- مرجع سیاست دسترسی/پلن/لایسنس
- دریافت و پردازش رویداد مصرف سرویس‌ها
- صدور فرمان مرکزی برای تعلیق/فعال‌سازی/تغییر پلن

## قراردادهای وابسته
- مانیفست مخزن: `.abrak/manifest.yml`
- قرارداد سرویس: `.abrak/service-contract.yml`
- مرجع مرکزی رجیستری: `https://github.com/mostafazadeceo/abrak-co-dataenter-v1/tree/main/registry`

## قیود ایمنی
- هیچ عملیات اجرایی روی `PC.HAIDA.CO` و `CRM.HAIDA.CO`
- هر تغییر فقط با ADR و مسیر بازگشت
