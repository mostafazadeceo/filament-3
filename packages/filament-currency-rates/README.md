# افزونه نرخ ارز (فیلامنت)

این افزونه نرخ ارزهای کلیدی را هر ۳۰ دقیقه از سایت **الان‌چند** اسکرپ می‌کند و در دیتابیس ذخیره می‌نماید. همچنین یک ای‌پی‌آی داخلی برای دریافت نرخ‌ها به ریال یا تومان (براساس تنظیمات) در اختیار شما قرار می‌دهد و صفحه تنظیمات کامل دارد.

## امکانات
- اسکرپ خودکار نرخ‌ها از https://alanchand.com/
- زمان‌بندی پیش‌فرض: هر ۳۰ دقیقه
- ذخیره آخرین نرخ‌ها در جدول اختصاصی
- گزارش هر اجرای همگام‌سازی
- صفحه تنظیمات کامل در پنل فیلامنت
- ای‌پی‌آی داخلی برای دریافت نرخ‌ها
- ابزار کمکی برای تبدیل قیمت‌ها به ریال
- انتخاب واحد منبع و واحد نمایش (ریال/تومان)
- انتخاب مبنای نرخ (خرید/فروش/میانگین)
- اعمال سود درصدی و ثابت روی نرخ‌ها

---

## نصب

### ۱) افزودن پکیج (لوکال)
در ریشه پروژه:

```bash
composer update haida/filament-currency-rates
```

### ۲) اجرای مایگریشن‌ها

```bash
php artisan migrate
```

### ۳) ثبت افزونه در پنل فیلامنت
این مرحله از قبل در پروژه اعمال شده است.

## تنظیمات پنل
مسیر: **تنظیمات → تنظیمات نرخ ارز**

### بخش‌ها
- **وضعیت و زمان‌بندی**: فعال/غیرفعال کردن همگام‌سازی و تنظیم تایم رفرش (دقیقه)
- **منبع داده**:
  - حالت پیش‌فرض: اسکرپ از الان‌چند
  - حالت ای‌پی‌آی سفارشی: وارد کردن نشانی و توکن
- **واحد قیمت‌ها**:
  - **واحد داده منبع**: پیش‌فرض تومان (قابل تغییر به ریال)
  - **واحد نمایش**: نمایش نرخ‌ها در پنل و خروجی ای‌پی‌آی
- **قیمت‌گذاری**:
  - **مبنای نرخ**: خرید/فروش/میانگین
  - **سود درصدی** و **سود ثابت** (واحد ریال/تومان)
  - فرمول: نرخ نهایی = نرخ مبنا + (نرخ مبنا × درصد) + سود ثابت
- **ارزهای مورد نیاز**: تعیین کد ارزها (پیش‌فرض: usd, eur, gbp, aed, cny)
- **ای‌پی‌آی خروجی**: فعال‌سازی ای‌پی‌آی و تعیین توکن دسترسی
- **پیشرفته**: کش، تایم‌اوت و تعداد تلاش مجدد

---

## مسیرهای ای‌پی‌آی
به‌صورت پیش‌فرض فعال هستند. اگر توکن تعیین کنید باید هدر ارسال شود.
خروجی `buy` و `sell` با درنظرگرفتن سود تنظیم‌شده نمایش داده می‌شود و فیلدهای `buy_raw` و `sell_raw` مقدار خام را نگه می‌دارند.

### دریافت همه نرخ‌ها
```
GET /api/currency-rates
```

### دریافت نرخ یک ارز
```
GET /api/currency-rates/{code}
```
مثال:
```
GET /api/currency-rates/usd
```

### توکن دسترسی (اختیاری)
اگر در تنظیمات توکن ای‌پی‌آی تعیین شود:
```
X-Rate-Token: توکن_شما
```

## استفاده در کدنویسی (برای قیمت‌گذاری)

```php
use Haida\FilamentCurrencyRates\Services\CurrencyRateManager;

$manager = app(CurrencyRateManager::class);

// نرخ فروش دلار به ریال
$usdSell = $manager->getSellPrice('usd');

// تبدیل 10 دلار به ریال (با درنظر گرفتن مبنا و سود)
$priceIrr = $manager->convertToIrr(10, 'usd');

// نرخ نهایی بر اساس مبنای انتخابی و سود
$effective = $manager->getEffectiveRate('usd');
```

## نکات مهم
- اگر صفحه منبع تغییر ساختار بدهد، ممکن است نیاز به به‌روزرسانی پارسر داشته باشد.
- برای حجم بالاتر یا دقت بیشتر، می‌توانید منبع را به ای‌پی‌آی سفارشی تغییر دهید.
- وقتی سود فعال باشد، خروجی پنل و ای‌پی‌آی بر اساس سود محاسبه می‌شود.
